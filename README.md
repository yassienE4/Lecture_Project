Minecraft Themed Platformer

Controls:

Move Right: D/right arrow

Move Left: A/Left Arrow

Jump: W/Space/Up Arrow

Charge Bow: Hold C

Shoot arrow: release C after charge (1000ms)

Pause: Esc
